# Overview

**s-blacklist** is an underground street racing system for FiveM, inspired by NFS: Most Wanted.
Build a reputation, climb the tiers, fight for the **Blacklist Top 10**, wager cars in **Pink
Slips**, dodge **police Heat**, take **bounties**, and run **Race Hubs** — all from a premium
in-game UI (the **Street Tablet** and a custom radial).

## Highlights

- **Server-authoritative** racing (OneSync) — winners are decided server-side, not by the client.
- **Standalone-first**: works with no framework, or bridges to **QBCore / ESX / Qbox / ox**.
- **No bundled MDT**: dispatch is emitted to *your* police system.
- **Access control**: not everyone can use it — configurable requirements + optional whitelist.
- **Extensible & encryptable**: integrate your own scripts via exports/events without touching the core.
