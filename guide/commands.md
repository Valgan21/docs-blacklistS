# Commands

## Player

| Command | What it does |
|---|---|
| `/race [id] [buyin] [mods]` | Challenge a nearby player to a Sprint |
| `/chase [id] [buyin]` | Challenge to a Chase (cat & mouse) |
| `/raceaccept` | Accept the pending race/chase challenge |
| `/endurance [id] [buyin]` · `/enduranceaccept` | 3-leg Endurance |
| `/pinkslip [id] [plate] [mode]` · `/pinkslipaccept [plate]` | Wager your car |
| `/signup [raceId]` | Sign up for a scheduled hub race |
| `/bounty [id] [amount]` | Post a bounty on a player |
| `/tracker` | Toggle map visibility (+10% REP while on) |
| `/alias [name]` | Set/change your racing alias |
| `/title [name]` · `/iconic [plate]` · `/prestige` | Titles, iconic vehicle, prestige |
| `/trackrace [trackId] [id]` · `/trackrate [id] [1-5]` · `/tracks` | Community tracks |
| `/sbtop10` · `/payheat` | Quick Blacklist Top 10 · reduce Heat |

::: tip Radial menu
Press **M** (configurable) for the radial: challenges (with buy-in / Pink Slip confirmation),
Street Tablet, tracker, lower heat, nearest hub, Top 10 and perks.
:::

## Admin

| Command | What it does |
|---|---|
| `/s-blacklist:debugbridge` | Report the active bridge & its 14 functions |
| `/s-blacklist:setrep [id] [amount]` | Set a player's REP (jump tiers) |
| `/s-blacklist:event:start [id]` | Force a weekly event |
| `/s-blacklist:season:advance` | Advance the season state (testing) |
| `/s-blacklist:achievement:grant [id] [achId]` | Grant an achievement |
