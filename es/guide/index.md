# Resumen

**s-blacklist** es un sistema de carreras clandestinas para FiveM, inspirado en NFS: Most Wanted.
Construí tu reputación, subí de tier, peleá por el **Blacklist Top 10**, apostá coches en **Pink
Slips**, esquivá el **Heat policial**, tomá **bounties** y corré en los **Race Hubs** — todo desde
una UI premium in-game (el **Street Tablet** y un radial propio).

## Lo clave

- Carreras **server-authoritative** (OneSync) — el ganador lo decide el servidor, no el cliente.
- **Standalone-first**: funciona sin framework, o con bridge a **QBCore / ESX / Qbox / ox**.
- **Sin MDT propio**: el dispatch se emite a *tu* sistema policial.
- **Control de acceso**: no todos pueden usarlo — requisitos configurables + whitelist opcional.
- **Extensible y encriptable**: integrá tus scripts vía exports/eventos sin tocar el core.
