# Comandos

## Jugador

| Comando | Qué hace |
|---|---|
| `/race [id] [buyin] [mods]` | Retar a un jugador cercano a un Sprint |
| `/chase [id] [buyin]` | Retar a un Chase (gato y ratón) |
| `/raceaccept` | Aceptar el reto de carrera/chase pendiente |
| `/endurance [id] [buyin]` · `/enduranceaccept` | Endurance de 3 tramos |
| `/pinkslip [id] [placa] [modo]` · `/pinkslipaccept [placa]` | Apostar tu coche |
| `/signup [raceId]` | Inscribirte a una carrera programada del hub |
| `/bounty [id] [monto]` | Poner una recompensa sobre un jugador |
| `/tracker` | Visibilidad en el mapa (+10% REP mientras está activo) |
| `/alias [nombre]` | Poner/cambiar tu alias |
| `/title [nombre]` · `/iconic [placa]` · `/prestige` | Títulos, iconic vehicle, prestige |
| `/trackrace [trackId] [id]` · `/trackrate [id] [1-5]` · `/tracks` | Tracks comunitarios |
| `/sbtop10` · `/payheat` | Top 10 rápido · bajar Heat |

::: tip Radial
Tecla **M** (configurable) para el radial: retos (con buy-in / confirmación de Pink Slip),
Street Tablet, tracker, bajar heat, hub cercano, Top 10 y perks.
:::

## Admin

| Comando | Qué hace |
|---|---|
| `/s-blacklist:debugbridge` | Reporta el bridge activo y sus 14 funciones |
| `/s-blacklist:setrep [id] [monto]` | Setea la REP de un jugador (saltar tiers) |
| `/s-blacklist:event:start [id]` | Forzar un evento weekly |
| `/s-blacklist:season:advance` | Avanzar el estado de la season (testing) |
| `/s-blacklist:achievement:grant [id] [achId]` | Otorgar un logro |
